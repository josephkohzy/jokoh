PROJECT DETAILS

Shaco: Cat in the Box

CLIENT: Riot Games
YEAR: 2024
ROLE: Character Design & Animation
Project Description:

Complete character design and animation concept for TFT Tactician featuring chibi-style character art, environment design, and 10-second victory finisher storyboard. This project involved creating a whimsical and engaging character that fits seamlessly into the Teamfight Tactics universe while maintaining a unique personality and memorable visual identity.

The design process included extensive research into existing TFT aesthetics, multiple character iterations, and collaborative feedback sessions with the development team to ensure the final design met both artistic and technical requirements.

Deliverables:
  • Character concept art sheets
  • Chibi-style character renders
  • Environment design layouts
  • 10-second victory finisher storyboard
  • Animation reference guide
  • Color palette documentation
